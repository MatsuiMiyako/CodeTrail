🐍 Python Lesson 2: If Statements & Logic

📘 What You’ll Learn

  • How to use if, elif, and else
  • How to compare values using conditions
  • Logical operators: and, or, and not

✅ 1. Basic If Statements

age = 16
if age >= 18:
    print("You're an adult!")
else:
    print("You're a minor.")

The code runs different blocks depending on the condition.

🔄 2. elif and else

score = 85

if score >= 90:
    print("A")
elif score >= 80:
    print("B")
elif score >= 70:
    print("C")
else:
    print("F")

elif lets you check multiple conditions in order.

🧠 3. Logical Operators

  • and: both must be true
  • or: at least one must be true
  • not: flips true/false
age = 16
has_permission = True

if age >= 13 and has_permission:
    print("Access granted!")

🔍 4. Comparison Operators

  • ==: equal
  • !=: not equal
  • >, <, >=, <= 
num = 10
if num != 5:
    print("Not five!")

🧪 Practice Time!

Write a script that:

  1. Asks the user for a number
  2. Prints whether it's positive, negative, or zero
num = int(input("Enter a number: "))
if num > 0:
    print("Positive")
elif num < 0:
    print("Negative")
else:
    print("Zero")

🎁 Bonus Tip

You can combine logic with user input:

username = input("Enter username: ")
if username != "":
    print("Welcome, " + username)
else:
    print("You didn't enter a name!")

🧠 Quick Quiz

  1. What does elif mean?
  2. What’s the difference between and and or?
  3. How do you check if two values are not equal?
  4. What happens if no condition is true?
📝 Example answers:
1. "Else if" — a second condition to check.
2. and needs both true; or only needs one.
3. Use !=
4. The else block runs (if there is one).
